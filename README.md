# AcamarWeb---Sistema-de-Avalia--o-Digital-Escolar
 
O processo de avaliação em um sistema educacional é uma parte importante e primordial do seu sucesso para garantir a maneira correta de transmissão do conhecimento e para garantir que os alunos estejam trabalhando corretamente e tenham sucesso para adquirir o conhecimento necessário. Muitos métodos de avaliação podem ser utilizados: realização de algumas experiências, realização de diferentes miniprojetos, realização de testes e exames. O mecanismo de avaliação contribui para o processo de obtenção de informações acerca dos resultados esperados do aluno, bem como também se esses resultados, estão em conformidade com a forma como os professores ensina.<br>
A avaliação para a aprendizagem é uma abordagem ao ensino e à aprendizagem que cria feedback que é usado para melhorar o desempenho dos alunos. Os alunos se envolvem mais no processo de aprendizagem e, com isso, ganham confiança no que se espera que aprendam e em que padrão. Uma maneira de pensar sobre a avaliação é que ela visa "fechar a lacuna" entre a situação atual de um aluno e o local em que ele deseja estar na sua aprendizagem e conquista. <br>
O referido trabalho foi desenvolvido para a escola CETI – Hesíchia de Sousa Brito, onde na ocasião, fora destinado a realização de avaliações. Os estudantes realizavam suas provas através de seus dispositivos celulares e computadores de suas casas. Posteriormente, o projeto passou por modificações, ganhando uma versão Desktop para ser usado no laboratório de informática na própria instituição, o projeto ficou batizado de Acamar Desktop e é utilizado ate dia de hoje. <br>
O projeto foi descontinuado, uma vez que a escola optou por usar apenas a versão desktop por questões de controle interno. Mais tarde, com a chegada da pandemia de covid-19, a escola optou por usar uma plataforma virtual, eEscolar, e passou a utilizar um sistema nativo de avaliação escolar que já vinha na plataforma, contudo, esta ferramenta teve como base boa parte do código fonte da versão 1.0 do Acamar Web. <br>
#Descrição do projeto
O sistema conta com uma versão de Administrador que pode ser usado tanto pela gestão da escola quanto pelos professores, e uma versão para estudantes. Na tela inicial, essas opções podem ser acessadas com dois logins distintos neste painel. Além de ainda, possuir uma opção de cadastro de novo usuário. <br>
